## Add your images
Place transparent PNGs in `public/imgs/f250/` using these exact names:
- base_top.png, base_driver.png, base_passenger.png, base_front.png, base_rear.png
- workbox_top.png, workbox_driver.png, workbox_passenger.png, workbox_front.png, workbox_rear.png

They should all share the same canvas dimensions.
